# Getting Started

### Preparation

- Create New Project
- Create Empty Repository
- Run `git init`, `git remote add origin URL`, `git checkout -b master`
- Run `npm init` and fill in details

### Install Gardener

    # npm install --save-dev grunt-gardener

### Run Basic Setup

    # grunt gardener

This will generate some files.
